material-ui-country-flags
=========================

Library with country flag icons for Material-UI.

## Installation

```shell
  npm install material-ui-country-flags --save
```

## Usage

```js
  var icon = require('material-ui-country-flags')
```

## Tests

```shell
   npm test
```
